
## Listport
Chromium extension for exporting large reading lists 

<ol>
  <li>Download the zip and unpack it
</li>
  <li>Go to chrome://extensions or brave://extensions
</li>
  <li>Enable Developer mode (top right)
</li>
  <li>Click "Load unpacked" and select the folder
</li>
</ol>

### Features
- Export as JSON or HTML
- Filter by read / unread, site, keyword, date range etc
- Sort by date or title
- Copy Export types straight to clipboard

### Permissions
Requests readingList only. completely local, no access needed or website interaction.

### Notes
if the popup shows an error, check your browser version first. make sure its above chrome 120+
Built for personal use, shared in case useful
